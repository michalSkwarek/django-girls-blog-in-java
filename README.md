# django-girls-blog-in-java
Blog platform from a tutorial https://tutorial.djangogirls.org/pl/ moved to the Java / JS technologies
### Technologies
- Java 7
- Spring 4 (Boot, MVC, Security, Test, Data)
- Hibernate 5
- JUnit, Mockito
- HTML5, CSS3, Bootstrap
- SQL, MySQL, HSQLDB
- JavaScript, Angular 1
- Lombook, Maven
